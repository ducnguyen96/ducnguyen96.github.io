Full Text Queries

Match query​

GET /recipe/_search
{
  "query": {
    "match": {
      "title": "Recipes with pasta or spaghetti"
    }
  }
}

Matching phrase​

GET /recipe/_search
{
  "queyr": {
    "match_phrase": {
      "title": "spaghetti puttanesca"
    }
  }
}

Searching multiple fields​

GET /recipe/_search
{
  "queyr": {
    "multi-match": {
      "query": "pasta",
      "fields": ["title", "description"]
    }
  }
}
